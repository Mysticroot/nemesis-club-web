API Documentation

RESTful API for managing administrators, blog posts, and sponsorships

https://yourdomain.com/api
Admin Management Blog Management Sponsor Management

Admin Management

Method Endpoint Auth Description
POST /api/admin/register Public
Register a new administrator account
{ "name": "John Doe", "email": "john@example.com", "password": "securePassword123" }
POST /api/admin/login Public
Authenticate admin and receive JWT token
{ "email": "john@example.com", "password": "securePassword123" }
GET /api/admin/ Bearer Token
Retrieve list of all administrators
GET /api/admin/:id Bearer Token
Get specific administrator by ID
DELETE /api/admin/me Bearer Token
Delete current admin account (irreversible)
DELETE /api/admin/:id Super Admin
Delete administrator by ID (super admin only)

Blog Management

Method Endpoint Auth Description
GET /api/blogs/ Public
Retrieve all published blog posts
GET /api/blogs/:id Public
Get specific blog post by ID
POST /api/blogs/ Bearer Token
Create new blog post
{ "title": "How Gears Work", "content": "This is a detailed blog about gears..." }
PUT /api/blogs/:id Bearer Token
Update existing blog post
DELETE /api/blogs/:id Bearer Token
Delete blog post permanently

Sponsorship Management

Method Endpoint Auth Description
POST /api/sponsors/submit Public
Submit new sponsorship request
{ "name": "Alice Johnson", "company_name": "TechCorp Solutions", "email": "alice@techcorp.com", "phone": "1234567890", "message": "We'd love to discuss sponsorship opportunities." }
GET /api/sponsors/requests Bearer Token
Retrieve all sponsorship requests
PUT /api/sponsors/requests/:id/status Bearer Token
Update sponsorship request status
{ "status": "approved" }
GET /api/sponsors/requests/approved Bearer Token
Get all approved sponsorship requests

History Management

Method Endpoint Auth Description
GET /api/history/ Public
View all history entries
GET /api/history/:year Public
View history entries by specific year
POST /api/history/ Bearer Token
Create new history entry (admin only)
DELETE /api/history/:id Bearer Token
Delete history entry by ID (admin only)
